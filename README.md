# Brain-Tumor-Segmentation
Compares multiple U-Net architectures and proposes an enhanced Attention-Inception U-Net for accurate tumor segmentation in MRI images.

# Brain Tumor Segmentation using U-Net Variants

## 📌 Problem
Segment tumor regions from brain MRI images.

## ⚙️ Tech Stack
- Python
- Deep Learning (U-Net, Attention U-Net, Inception U-Net)

## 🚀 Approach
- Compared four U-Net architectures
- Developed Attention-Inception U-Net for improved performance

## 📊 Results
- Achieved superior segmentation accuracy with Attention-Inception U-Net

## 💡 Impact
Supports automated tumor detection and medical diagnosis.
